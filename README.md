## Loan Approval Prediction System
# Project Overview
The Loan Approval Prediction System is a Machine Learning web application that predicts whether a loan will be approved or not approved based on applicant details. The system uses a Support Vector Machine (SVM) model trained on a loan dataset and provides predictions through a Flask-based web interface.

This project demonstrates how Machine Learning models can be integrated with a web application to make real-time predictions.
## Features

- Predicts loan approval status
- User-friendly web interface
- Input form for applicant details
- Help page explaining each field
- Machine learning model using SVM
- Real-time prediction using Flask backend
- Clean and responsive UI with gradient background
